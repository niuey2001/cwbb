# cwbb

开发语言：VB  、数据库: ACCESS。

实战内容：F1BOOK控件、excel 操作、压缩加密、文件导出。

功能包括：用户注册、登陆、修改密码、填写报表、报表版本设置、报表审核，加载上期数据、历史数据管理、操作说明等。
